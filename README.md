# Seazone — Guia Digital do Hóspede

Teste técnico Seazone (AI Builder). Guia digital personalizado por imóvel com IA.

## Stack

- Next.js 14 (App Router) + TypeScript strict
- Tailwind CSS + shadcn/ui
- Drizzle ORM + Postgres (Render)
- Anthropic SDK + Vercel AI SDK (streaming)
- Vitest

## Setup

```bash
cp .env.example .env.local  # preencha as keys
npm install
npm run dev
```

## Scripts

- `npm run dev` — dev server
- `npm run build` — produção
- `npm test` — Vitest
- `npm run db:generate` / `db:push` / `db:studio` — Drizzle
